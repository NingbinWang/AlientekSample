# STM32Sample
在这个仓库中，主要是将我以前用过的开发板再拯救起来，做一些我以前没有开发过的一些事情

# 功能设计
* 1.接入OV7670的sensor,接入到图像，输出给TFT_LCD的屏幕。
* 2.将MPU6050的数据抓到做相关的解析
* 3.引入lvgl做相关的GUI
* 4.使用MQTT的协议做相关的数据
